# Diagramas

Esta carpeta contiene dos tipos de archivos:

- `*.svg`: imagenes renderizadas para visualizar los diagramas sin depender del soporte Mermaid del visor
- `src/*.mmd`: fuentes Mermaid a partir de las que se generan los SVG

## Diagramas actuales

- `architecture-data-flow.svg`
- `architecture-environments.svg`
- `database-overview.svg`
- `database-core.svg`
- `database-competitive.svg`
- `database-user-auth.svg`

## Regeneracion

Los SVG se pueden volver a generar con Mermaid CLI a partir de los archivos de `src/`.
